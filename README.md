# Java Invoice Generator 📄

A professional console-based application designed to help freelancers and service providers generate quick, formatted invoices for their clients.

## ✨ Features
*   **Dynamic Input:** Captures client details and project names in real-time.
*   **Cost Calculation:** Automatically calculates the total amount due based on hours worked and hourly rates.
*   **Status Tracking:** Includes payment status tracking (Pending/Paid).
*   **Professional Formatting:** Generates a structured, easy-to-read "Official Invoice" layout in the console.

## 🚀 How to Use
1. Clone the repository.
2. Compile the Java file: `javac BillingSystem.java`
3. Run the application: `java BillingSystem`
4. Follow the on-screen prompts to enter project details.

## 🛠️ Technical Details
*   **Language:** Java
*   **Key Concepts:** Scanner I/O, Arithmetic Operations, Variable Management (double, int, boolean), and Input Buffer handling.
*   **Architecture:** Logic-based procedural programming.
